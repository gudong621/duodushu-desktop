为什么选择桌面版？

绿色便携 (Portable)

无需安装，解压即用。即插即用的 U 盘模式，让你的书库、笔记和学习记录随身携带，换电脑也无需迁移。

100% 隐私安全

你的书籍、笔记、阅读数据完全存储在本地。没有云端上传，没有后台追踪，完全由你掌控。

BYO AI Key

支持绑定你自己的 API Key。按需付费，直连服务商，享受无限量的 AI 翻译与深度解析。

即点即查

阅读时不被打断。点击任意单词即刻查看释义。支持记笔记、AI 深度解析，让生词不再是阅读的拦路虎。

MDX 词典支持

兼容你的专业词库。支持直接导入 MDX 格式词典，丰富的词典资源，按需加载。

场景记忆 & 例句库

看过的书就是你的例句库。复习单词时，系统会自动回溯该单词在你读过的书中出现的原句，帮你重现语境记忆。

Web 版 vs 桌面版

功能特性 Web/Docker 版 桌面便携版
